# CoinVault

**CoinVault** is a modern cryptocurrency tracker built with **React 19** and **Vite**. This project is designed for real-time market monitoring and personal portfolio management.

## Technologies
* **Frontend:** React 19, Vite
* **Styling:** CSS3 (Modern Dark Theme)
* **API:** CoinGecko (Planned)
* **Backend:** Supabase (Planned)
